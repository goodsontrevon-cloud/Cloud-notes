# Git & Github basics

---

## What is git?
Git is a version control system that tracks changes in code.
Git is a version control system that tracks changes in files over time.

---

## Common Commands
- git init
- git add  .
- git commit -m "message"
- git push
- `git clone` = copy a repo to your computer
- `git status` = check what changed
- `git add .` = stage changes
- `git commit -m "message"` = save changes
- `git push` = upload to Github

---

## What is Github?
GitHub is where you store and share your code online.
GitHub is a platform where you store and share your Git repositories online.
