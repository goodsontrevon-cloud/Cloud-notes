# Linux Basics

## Common Commands

- pwd :arrow_right: shows current directory
- ls :arrow_right: lists files
- cd :arrow_right: change directory
- mkdir :arrow_right: create folder 
- touch :arrow_right: create file
- rn :arrow_right: delete file
- cp :arrow_right: copy file
- mv :arrow_right: move/rename file
