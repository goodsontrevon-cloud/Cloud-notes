# Linux Basics
---

## What is Linux?
Linux is an operating system commonly used on servers in cloud environments.

---

## Why Linux matter in cloud
Many cloud servers run on Linux, so learning basic commands is important for cloud engineering.

---

## Common Commands

- `pwd` :arrow_right: shows current directory
- `ls` :arrow_right: lists files
- `cd` :arrow_right: change directory
- `mkdir` :arrow_right: create folder 
- `touch` :arrow_right: create file
- `rn` :arrow_right: delete file
- `cp` :arrow_right: copy file
- `mv` :arrow_right: move/rename file
- `rm` :arrow_right: deletes a file
-  `cat` :arrow_right: shows file contents
