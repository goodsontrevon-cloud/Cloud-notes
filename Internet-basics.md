# Internet Basics

---

## What is an IP Address?
An IP address is a unique number used to indentify a device on a network.
An IP address is a number used to identify a device on a network.

---

## What is DNS?
DNS (Domain Name systen) translates domain names (like google.com) into IP addresses.
DNS translates website name like google.com into IP addresses.

---

## What is a Service?
A server is a computer that provides services or data to other computers.
A server is a computer that provides data or services to other devices.

---

## What is a Client?
A client is a device (like your phone or laptop) that requests data from a server.

---

## What I learned
The internet works by devices communicating with each other using addresses and services.
